## **About**

Process orders from a flower shop

## Setup

yarn:

```sh
# Install dependencies
yarn

# Start nodemon (view example output)
yarn dev

# Run tests
yarn test

# Check types tests
yarn type-check

# Build to Javascript (output in ./dist)
yarn build

# Run compiled build (runs the example file to show output)
yarn start
```
